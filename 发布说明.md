# 发布成网页链接

这个目录已经是 Firebase Hosting 项目结构。

## 发布前准备

1. 安装 Node.js
2. 安装 Firebase CLI：

```bash
npm install -g firebase-tools
```

3. 登录 Firebase：

```bash
firebase login
```

## 发布

在这个目录执行：

```bash
firebase deploy --only hosting
```

目录路径：

```text
C:\Users\admin\Documents\Codex\2026-07-08\ban\outputs\wedding-memo-site
```

发布成功后，Firebase 会给出链接。根据当前项目 ID，常见链接会是：

```text
https://wedding-memo-e78c4.web.app
```

或：

```text
https://wedding-memo-e78c4.firebaseapp.com
```
