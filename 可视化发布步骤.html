备婚页面发布可视化步骤

目标：把本地的备婚 HTML 发布成一个可以手机、电脑访问的网页链接。
1. 打开项目
2. 打开终端
3. 登录 Firebase
4. 发布拿链接

一、在 Trae CN 打开项目

1

打开文件夹

在 Trae CN 顶部菜单找：文件 → 打开文件夹。

选择这个目录：

C:\Users\admin\Documents\Codex\2026-07-08\ban\outputs\wedding-memo-site
Trae CN
文件
打开文件夹...
选择 wedding-memo-site
2

确认项目结构

左侧文件列表应该能看到这几个文件：

public index.html firebase.json .firebaserc 发布说明.md
如果只看到一个 HTML 文件，说明打开错目录了，要打开 wedding-memo-site 文件夹。
文件列表
▾ wedding-memo-site
　▾ public
　　index.html
　firebase.json
　.firebaserc

二、打开终端并检查环境

3

打开终端

在 Trae CN 里找：终端 → 新建终端。

如果找不到，试试顶部菜单里的 Terminal 或底部面板。

Trae CN
终端
新建终端
PS C:\...\wedding-memo-site>
4

检查 Node 和 npm

在终端输入：

node -v npm -v
能显示版本号就继续。若提示找不到 node/npm，需要先安装 Node.js。
终端
PS> node -v v22.0.0 PS> npm -v 10.0.0
5

如果没有 Node.js

打开官网：https://nodejs.org/，下载安装 LTS 版本。

安装完成后，重启 Trae CN，再重新执行 node -v 和 npm -v。

Node.js 官网
Download Node.js LTS
安装完成后重启 Trae CN

三、安装并登录 Firebase

6

安装 Firebase CLI

在终端输入：

npm install -g firebase-tools

安装完成后检查：

firebase --version
终端
PS> npm install -g firebase-tools ... PS> firebase --version 13.x.x
7

登录 Firebase

输入：

firebase login

会弹出浏览器，用你创建 Firebase 项目的 Google 账号登录。

浏览器
Google 登录
允许 Firebase CLI 访问
看到 Success! 就可以回到终端。

四、发布网站并拿到链接

8

确认终端在正确目录

终端路径最后应该是 wedding-memo-site。

如果不是，输入：

cd C:\Users\admin\Documents\Codex\2026-07-08\ban\outputs\wedding-memo-site
终端路径
PS C:\Users\admin\Documents\Codex\2026-07-08\ban\outputs\wedding-memo-site>
9

发布

输入：

firebase deploy --only hosting

等待终端出现 Deploy complete!。

终端
✔ Deploy complete! Hosting URL: https://wedding-memo-e78c4.web.app
10

打开链接

复制终端里的 Hosting URL，在浏览器打开。

https://wedding-memo-e78c4.web.app

打开后点右上角 云端设置，填 Firebase 配置和文档 ID。

备婚页面
云端设置
文档 ID：wedding-2026-11-13
连接并保存

成功后应该看到什么

网页能打开

浏览器地址栏是 https://...，不是 file://...。

页面能保存

右上角显示“云端已保存”或“云端实时同步”。

Firestore 有数据

Firebase 控制台出现 weddingMemos / wedding-2026-11-13。
